1. The following describe-instances example uses filters to scope the results to specific instanceId  and the --query parameter to display only the instance IDs and availability zone.

aws ec2 describe-instances \
    --filters Name=instance-type,Values=t2.micro,t3.micro \
    --instance-ids i-0098335d1905a32c2 \
    --query 'Reservations[*].Instances[*].{Instance:InstanceId,AZ:Placement.AvailabilityZone}' \
    --output json
